## モデルハウス一覧

### HTTPメソッド
```
HTTP: GET 
```

### リクエスト

| パラメータ名 | 型 | 必須 | デフォルト値 | 説明 |
|------------|---|-----|------------|-----|
| `page` | integer | No | 1 | ページ番号（1以上） |
| `per_page` | integer | No | 50 | 1ページ当たりの件数（1-100） |
| `keyword` | string | No | "" | フリーワード検索（スペース区切りでAND検索） |
| `company_id` | integer | No | "" | 企業ID（空なら全件、IDがあれば該当企業のみ） |
| `is_active` | integer | No | "" | 0: 非公開, 1: 公開, 空: 全件 |
| `order_by` | string | No | DESC | ソート順（ASC: 昇順、DESC: 降順） |

| パラメータ名 | 型 | 必須 | デフォルト値 | 説明 |
|------------|---|-----|------------|-----|
| `year_month` | string | No | 現在の年月 |  |
| `range` | integer | No | 3 | 前後○○ヵ月のデータを取得 |

### HTTPメソッド
```
HTTP: GET 
```

### リクエスト

## [物件管理] 見学予約日の登録 [一括登録]

### HTTPメソッド
```
HTTP: POST
```

### 補足

```
フロントからは日付の範囲選択にてデータをリクエストします
API側で日付範囲分のデータを挿入してください

また、既に同一の日付でデータが存在する場合、上書き保存とする

"start_date": "2025/09/12"
"end_date": "2025/09/30"

2025/09/12
2025/09/13
2025/09/14
2025/09/15
*
*
*
2025/09/30
```

### 補足2

```
保存データが大量となり（例えば2026/01/01 ~ 2026/12/31 指定された場合）、timeoutしてしまうなど、
問題が発生するようでしたら、フロント側で、当月から○ヵ月までしか選択できないようにします。
その際は、ご連絡ください。
```

### リクエスト
```json
{
    "start_date": "2025/09/12",
    "end_date": "2025/09/30",
    "status": 1,
    "time_slots": {
        "start_time": "09:00",
        "end_time": "18:00",
        "is_main": 0,
    }
},
```

### レスポンス
```json
// 正常
HTTP:201
{
    "success": true,
    "message": "",
    "data": {
        "id": [
            1000,
            1001,
            1002
        ]
    }
}
```

```json
// エラー時
{
    "success": false,
    "message": "",
    "errors": {}
}
```
